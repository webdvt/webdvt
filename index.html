About Us WebDVT


We are a student organization at Virginia Tech who is dedicated to teaching web development to Hokies. We hold weekly workshop incorporating lessons as well as hands-on experience such as building web applications. The goal of WebDVT is to help members be able to build side-projects on their own.

Meet the team

Why Choose Us


Side-Project

We help members build small side-projects of their own.

Personal Website

Members will publish their own portfolio website by the end of the semester.

Version Control

We also teach version-control system like Git.

Practical

We teach practical usage of the web technologies so you can confidently use them in your projects.

What You Will Learn


Stunning Website

  • Build simple applications using vanilla JavaScript
  • Create websites using HTML, CSS, and JS
  • Host your site on Github pages

Complex Web App

  • Use Node.js to build backend API
  • Connect with external database like MongoDB
  • Create front-end UI using React/Redux
  • Publish to cloud platform like Heroku or AWS

Web API

  • Use Node.js with Express
  • Create HTTP Request Routes like GET and PUT
